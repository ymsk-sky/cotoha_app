Qiita記事にあげる。豪華景品ありらしい。

## アイデア

### ネタ系

- 2chコピペ解析（ルイズ、キリトなど）

- あいまいキーワードからVTuber推定
- AとA'の違いはなんですか？類似度
- だじゃれ解析
- 背後で爆発音がして振り返った
- ツンデレの感情
- もし俺が謝ってこられてたとしたら
- 対になる存在の類似度（逆だったかもしれねェ）
- フォレの名前は〜の聞き取り修正
